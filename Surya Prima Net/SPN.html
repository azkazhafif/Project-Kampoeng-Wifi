Tentang Kami

Kenapa memilih kami?

Surya Prima Net dengan bangga menyajikan layanan internet dan solusi teknologi terbaik dengan fokus utama pada kualitas dan kepuasan pelanggan. Kami terus berkomitmen untuk meningkatkan infrastruktur jaringan, mengadopsi teknologi terkini, dan memberikan dukungan teknis yang efisien. Sebagai perusahaan progresif, kami juga berpartisipasi aktif dalam program komunitas untuk memperluas akses internet. Dengan semangat inovasi, Kampoeng WiFi menjadi mitra terpercaya dalam memenuhi kebutuhan konektivitas di era digital.

Layanan Kami

Internet Service Provider

Sebagai penyedia layanan internet, kami menawarkan akses jaringan ke internet untuk perorangan dan perusahaan, serta layanan terkait seperti akses VPN, pembuatan situs web, dan virtual hosting. Dengan dukungan IP Public, IP Static, IP Private, serta penawaran Free Trial selama 7 hari, kami menjamin SLA 98%, dan menyediakan dukungan pelanggan 24/7.

Server Data Center

Fasilitas ini dirancang untuk menempatkan sistem komputer beserta komponen terkaitnya, seperti sistem telekomunikasi dan penyimpanan data. Biasanya, fasilitas ini juga mencakup catu daya dan koneksi komunikasi data yang redundan, serta perangkat pengontrol dan keamanan fisik.

Network Configuration

Memanfaatkan seluruh kapasitas bandwidth dari jaringan seluler (SPN) dan telah terintegrasi dengan beberapa server, termasuk GGC, grup Facebook, CDN Akamai lokal, CDN Netflix lokal, server game Mobile Legends, Garena, Microsoft, Amazon, Cloudflare, serta akan terus bertambah dengan penambahan server di masa depan.
